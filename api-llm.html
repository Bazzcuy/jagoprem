example.py
from openai import OpenAI

client = OpenAI(
  api_key="sk-jgp-your-api-key",
  base_url="https://api.jagoprem.shop/v1"
)

response = client.chat.completions.create(
  model="openai/gpt-4o-mini",
  messages=[{"role":"user","content":"Halo!"}]
)

print(response.choices[0].message.content)

Mulai dalam tiga langkah

Alurnya dibuat singkat untuk developer maupun pengguna yang baru pertama memakai API.

1

Pilih paket token

Beli mulai dari 1 juta token melalui toko JagoPrem dan lakukan pembayaran QRIS.

2

Terima API key

API key dan informasi akun dikirim ke WhatsApp aktif setelah pembayaran dikonfirmasi.

3

Hubungkan aplikasi

Ganti base URL pada SDK OpenAI, pilih model, lalu mulai melakukan request.

Pertanyaan umum

Apa itu paket token JagoPrem API?

Paket token adalah kuota penggunaan API. Saldo berkurang mengikuti token input dan output dari setiap request.

Apakah paket memiliki masa aktif?

Paket bersifat pay-as-you-go dan dapat digunakan selama akun aktif serta masih memiliki saldo token.

Bagaimana API key dikirim?

Setelah pembayaran QRIS dikonfirmasi, API key dikirim ke nomor WhatsApp yang diisi saat checkout.

Apakah kompatibel dengan SDK OpenAI?

Ya. Cukup ubah base URL menjadi endpoint JagoPrem dan gunakan API key yang diberikan.

Siap membangun dengan AI?

Mulai dari 1 juta token dan gunakan model yang paling sesuai dengan produkmu.
