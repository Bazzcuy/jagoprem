About Agents

Agents allow you to create autonomous AI workflows that can browse the web, write code, manage files, and call APIs. They run on top of your JagoPrem virtual key and use your allocated budget.

What is MCP?

MCP (Model Context Protocol) lets you connect external tools and data sources to your AI models. Examples: web search, database access, file systems, and custom APIs. Contact JagoPrem admin to configure MCP server access.
