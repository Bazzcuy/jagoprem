Pengenalan

Selamat datang di dokumentasi resmi JagoPrem LLM API. API ini dirancang untuk memberikan Anda akses instan ke 100+ model AI (seperti GPT-4o, Claude 3.5, Gemini 1.5, DeepSeek, dll) melalui satu endpoint tunggal yang 100% kompatibel dengan format OpenAI API.

Anda tidak perlu mengubah kode (jika sudah menggunakan OpenAI SDK), cukup ganti Base URL dan API Key Anda dengan kredensial dari JagoPrem LLM.

Info Sistem
JagoPrem LLM menggunakan model pay-as-you-go. Saldo Anda dipotong sesuai pemakaian token per request. Tidak ada masa kadaluarsa (expired) untuk saldo.

Pembuatan Akun & API Key

Langkah 1: Membeli Saldo API

Pembelian saldo dilakukan langsung melalui platform JagoPrem Store. Pilih kategori API LLM, lalu pilih paket saldo yang Anda butuhkan (mulai dari $5).

Kurs yang berlaku adalah tetap: $1 USD = Rp 12.300.

Langkah 2: Proses Aktivasi

Setelah melakukan pembayaran via QRIS/e-Wallet di JagoPrem Store, admin kami akan memvalidasi pesanan Anda dan mengaktifkan akun Anda secara manual demi keamanan.

Langkah 3: Menerima API Key

Anda akan menerima API Key yang diawali dengan sk-jgp-... beserta informasi login dashboard (jika diperlukan) melalui WhatsApp atau email yang terdaftar saat checkout. Simpan API Key ini baik-baik dan jangan pernah membagikannya secara publik.

API Reference

JagoPrem LLM bertindak sebagai proxy server standar yang mengutamakan kompatibilitas dengan ekosistem OpenAI.

Base URL

https://api.jagoprem.shop/v1

Autentikasi

Gunakan API Key yang diberikan admin di header Authorization.

Authorization: Bearer sk-jgp-YOUR_API_KEY

Chat Completions

Endpoint utama untuk percakapan dengan model teks AI.

POST /chat/completions

Contoh Request (cURL)

curl https://api.jagoprem.shop/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-jgp-YOUR_API_KEY" \
  -d '{
    "model": "openai/gpt-4o-mini",
    "messages": [
      {
        "role": "system",
        "content": "Anda adalah asisten virtual yang sangat membantu."
      },
      {
        "role": "user",
        "content": "Halo! Siapa Anda?"
      }
    ]
  }'

Format Parameter Model

Karena kami mendukung berbagai provider, disarankan untuk menyertakan prefix provider di depan nama model (misal: openai/, anthropic/, google/).

Embeddings & Models

Embeddings

Ubah teks menjadi vektor representasi numerik untuk keperluan semantic search atau RAG.

POST /embeddings 
curl https://api.jagoprem.shop/v1/embeddings \
  -H "Authorization: Bearer sk-jgp-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/text-embedding-3-small",
    "input": "Kalimat yang ingin diubah menjadi vektor"
  }'

Daftar Model (Models API)

Melihat daftar model yang tersedia dan memiliki akses.

GET /models 
curl https://api.jagoprem.shop/v1/models \
  -H "Authorization: Bearer sk-jgp-YOUR_API_KEY"

Images & Audio

Image Generation

Endpoint untuk membuat gambar baru dari teks (contohnya menggunakan DALL-E 3 atau Stable Diffusion).

POST /images/generations 
curl https://api.jagoprem.shop/v1/images/generations \
  -H "Authorization: Bearer sk-jgp-YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/dall-e-3",
    "prompt": "Sebuah kota futuristik yang melayang di atas awan saat matahari terbenam",
    "n": 1,
    "size": "1024x1024"
  }'

Audio Transcriptions (Speech-to-Text)

Mengubah audio MP3/WAV menjadi teks menggunakan Whisper.

POST /audio/transcriptions

Python (OpenAI SDK)

JagoPrem LLM dapat digunakan secara drop-in dengan library resmi OpenAI untuk Python.

pip install openai
from openai import OpenAI

client = OpenAI(
    api_key="sk-jgp-YOUR_API_KEY",
    base_url="https://api.jagoprem.shop/v1"
)

response = client.chat.completions.create(
    model="anthropic/claude-3-5-sonnet",
    messages=[{"role": "user", "content": "Tuliskan puisi pendek."}]
)

print(response.choices[0].message.content)

VS Code (Cline)

Anda dapat mengonfigurasi AI assistant seperti Cline atau Cursor di text editor Anda.

  • Pilih provider API: OpenAI Compatible
  • Base URL: https://api.jagoprem.shop/v1
  • API Key: sk-jgp-YOUR_API_KEY
  • Model override: Ketik nama model misalnya anthropic/claude-3-5-sonnet
Perhatian: Pastikan jaringan internet Anda stabil jika request AI dari IDE karena timeout dapat menyebabkan koneksi terputus ke proxy server.
